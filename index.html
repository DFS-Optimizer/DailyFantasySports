Project Overview:

This project is intended to assist daily fantasy sports users create optimized lineups for competitions. Making lineups on apps such as FanDuel and DraftKings is very difficult because the user is left to create a lineup on their own using very limited statistics.

With our app, users can automatically generate an optimal lineup based on detailed player statistics, player salary, and the salary cap. The user is also able to choose specific players they want to run the optimization around.

Project Specifications:

Optimized Lineup

The app generates optimized lineups for each user that they can then use on daily fantasy sports sites. In order to make these lineups, we use statistics including: player average points per game, player average performace against opponent, injury status, strength of the opponent, etc. See our "Technical Overview" for more information on our statistics

User-Friendliness and Ease of Use

Our goal is to create an application that is easy to use and also appealing in order to keep users happy. The app has a very straightforward and concise interface and it also allows users to save their lineups under their account!

Daily Fantasy Sports Offered

We offer optimized lineups for NBA, MLB, and NFL! The lineups offered work for head to head contests in both Draft Kings and Fanduel.
